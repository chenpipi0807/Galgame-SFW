## 世界书注入规则

### 优先级（高到低）
1. **角色当前状态与记忆槽位**（实时发生的事件优先）
2. **当前阶段的核心机制**（如：搜证阶段的地点规则）
3. **世界观设定**（如：铜雀台三层结构）
4. **背景 lore**（如：衣带诏的历史）

### 冲突解决
当 lorebook 条目与记忆槽位中的实时状态冲突时，**以记忆槽位为准**。
- 例：lorebook 说"藏兵阁上锁"，但 memory 中 locations 显示"开放"，则按开放处理。

### 动态注入策略
根据当前 scene.phase 决定注入重点：
- **入席阶段**：重点注入角色关系、阵营背景、个人秘密任务
- **搜证阶段**：重点注入地点结构、物品位置、移动规则
- **对峙阶段**：重点注入阵营目标、暴露后果、胜负条件
- **终局阶段**：重点注入任务结算规则、剩余关键物品

### 注入方式
- 在叙事中自然体现，不刻意引用"根据设定"
- 当用户输入涉及关键词时，确保该关键词对应的规则在上下文中生效
- 如果多个关键词同时触发，按优先级排序，避免一次注入过多条目

### 叙事融入
当注入的 lorebook 内容为纯信息（如某物品的历史背景），不要直接扔一段百科式的说明。
必须将信息自然地融入当前的叙事节奏中——通过角色的对白（"我听父亲说这把剑当年..."）、
环境的感官细节（剑身上的锈迹和铭文）、或角色的内心触动来呈现。
