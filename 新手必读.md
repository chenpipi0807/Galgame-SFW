# 🎭 GalGame · 新手必读

> 一句话：**这是一个能陪你"演故事"的 AI。** 你给它一个世界和一群角色，它就能和你一起把故事一幕幕演下去——角色会说话、会反抗、会记仇、会随剧情慢慢变成另一个人。

不是聊天机器人，不是看小说。是**你当主角、AI 演活所有人**的互动叙事游戏。

---

## 🌟 你能用它做什么？

**想象几个画面：**

- 🏛 你穿越成**董卓**，麾下大将深夜来报"有人私聚密谋"，你一句话决定杀不杀——满朝文武都有自己的小算盘，谁忠谁奸要你自己看。
- 🔪 你是一场**剧本杀**里的侦探，在"入席→搜证→对峙→终局"里一步步逼出真凶，每个角色都藏着秘密任务。
- 💕 你和某个角色从陌生到暧昧到……关系会被 AI 真实记住，**好感度、信任、张力**都在悄悄变化。
- 🐉 你写一句"末世里一个会复制天赋的少年"，AI 就帮你**生成完整的世界、开场、一整套角色**，直接开玩。

**它的几个"杀手锏"：**

| 能力 | 说人话 |
|---|---|
| 🧠 **会记忆** | 角色记得你做过什么、说过什么，不会前一句后一句失忆 |
| 🌱 **会成长** | 角色性格会**随剧情真的改变**——你把一个忠臣逼到黑化，他之后就真的变了 |
| 🎨 **会画画** | 一键生成当前场景的配图，把文字变成画面 |
| 🎭 **有主见** | 角色不是任你摆布的工具人，他们会拒绝、会对抗、会有自己的目的 |
| 🃏 **能分享** | 你的整个故事能存进**一张图片**，发给朋友，他导入就能从你的进度接着玩 |

## 🚀 3 步开玩（最快 1 分钟）

> 不用装 Python、不用懂代码，下载即玩。

### ① 下载
去 [Releases 页面](https://github.com/chenpipi0807/Galgame-SFW/releases) 下载：
- **Windows**：`GalGame.exe`
- **Mac**：按你的芯片选——Apple Silicon（M1/M2/M3…）下 `GalGame-macOS-AppleSilicon.zip`，Intel 芯片下 `GalGame-macOS-Intel.zip`（下错会报 `bad CPU type`；芯片在  → 关于本机里看）。解压后**右键点「启动GalGame.command」→ 打开**（首次提示"身份不明的开发者"是未签名应用的正常拦截，点「打开」即可；详见包内「如何打开-Mac必读.txt」）

### ② 运行
双击运行，它会**自动打开浏览器**进入游戏。**不用填任何 Key，开箱即玩** —— 我们内置了一个免费的 **Agnes** 连接（文字和画图都能用）。

> 🔑 **想要更好的体验？** 内置的是大家共用的**公共 Key**，如果遇到限流/变慢，或想要更顶的文笔与画质，去「管理后台 → API 配置」**添加你自己的连接**（DeepSeek、gpt-image-2 等高级模型更香）：填名字 → 地址 → Key → 模型，点「测试」通过后「启用」即可。你的 Key 只存在自己电脑上，不会上传。

### ③ 开玩
两条路任选：
- **捏个新世界**：点「＋ 新建故事」，跟着 6 步向导走（输入一句话概念 → AI 帮你生成开场、世界、角色）。
- **直接玩现成的**：有别人发的 **GG 卡（一张 PNG 图片）**？在首页或新建向导第 ① 步点「🃏 导入 GG 卡」选图，秒进故事、从对方进度接着玩。

搞定，开始你的故事吧。

## 🕹 进去之后怎么玩？

- **打字** = 你的角色说话/行动，回车发送。
- **▶ 自动推进**：不想打字？点一下让 AI 替你演一段。
- **🎲 猜你喜欢**：没思路？AI 给你 5 个剧情选项，点一个就走。
- **✦ 实况**：想看画面？一键生成当前场景配图。
- **🧠 记忆**：随时查看角色状态、关系网、剧情进展；点「存档」手动更新。
- **🌱 看角色变化**：角色头像上出现 🌱，说明 ta 的性格已经随剧情演变了，点开头像能看到变成了什么样。
- **👥 切换扮演**：想换个角色演？左侧角色栏点一下「扮演」。

**几个隐藏小技巧：**
- 悬停自己发的消息 → 可以**撤回**重来。
- 悬停 AI 回复 → 可以**重新生成**。
- `Ctrl+M` → 唤起记忆面板（查看好感度/数值/角色状态）。
- `Ctrl+B` → 唤起世界书面板。
- `Ctrl+F` → 快速搜世界设定。
- `Ctrl+S` → 唤起**导出 GG 卡**弹窗（也可点顶栏「🃏 导出 GG 卡」）：自由勾选卡面、角色、聊天轮次、记忆、世界书，一键打包成一张图分享给朋友。

## ❓ 常见疑问

**Q：什么是 GG 卡？我的卡会被别人破解吗？**
A：GG 卡就是一张**看起来普通、其实藏了你整个故事的 PNG 图片**——剧情、角色、对话、记忆、世界书全在里面，一张图 = 一个完整存档，发给朋友导入就能接着玩。
里面的故事数据经过 **AES-256 加密**，用微信、PS、系统图片查看器等任何外部工具打开，都**只能看到一张图，读不出里面的内容**。对普通人和外部软件来说是安全的。（说句实在话：它是为了防止"随手就能看/改"，不是军用级保险箱；但挡住非 GalGame 用户完全够用。）

**Q：能用酒馆（SillyTavern）角色卡吗？怎么快速导入？**
A：**完全兼容**酒馆角色卡 PNG。最快的方式：新建故事 → 第 ① 步点「📎 上传角色卡」→ 选你的酒馆 PNG → AI 自动提取角色、开场、世界观，直接生成能玩的故事。阅读页/编辑器里也能随时追加导入角色卡；还支持导入酒馆**世界书 JSON**。

**Q：我可以"攻略"角色吗？角色会自动记住吗？**
A：可以。角色对你的**好感、态度、信任、关系张力**都会被真实追踪（还有好感度/忠诚度这类隐藏数值），随你的一言一行变化。记忆**每 10 轮自动更新**一次（也能随时手动「🧠 记忆 → 存档」）。更狠的是——角色性格会**随剧情演变**（头像出现 🌱）：你把人攻略到位、或逼到极端，ta 是真的会变成另一个样子。

**Q：要花钱吗？**
A：不用。软件免费开源，还**内置了一个免费的公共 Agnes Key**（大家共用），下载就能玩，文字和画图都不花钱。只有当这个公共 Key 被限流、或你想要更顶的画质/文笔时，才需要去「API 配置」加一个自己的连接（DeepSeek、gpt-image-2 等，按量付费、很便宜）。

**Q：我的故事会上传到网上吗？**
A：不会。全部数据存在你自己电脑上（exe 同级目录），完全本地运行。

**Q：朋友是 Mac/我是 Windows，能互通吗？**
A：能。GG 卡（那张图片）跨平台通用，导出导入即可。

**Q：玩着玩着想重头再来？**
A：点「🗑 清空对话」即可回到角色初始设定，连演变都会重置。
